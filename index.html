What can I do?

I can use HTML5, JavaScript, and CSS3/Bootstrap 5 to build web pages from scratch, or alter existing ones to fit whatever you might need.1

HTML5

Hypertext Markup Language 5 is the bones of a web page. It provides all the text and images. All of my work examples below include HTML5.

JavaScript

JavaScript is a scripting language commonly used with web pages to provide a more dynamic experience. Of my work examples below, Drive-Thru Calculator, To-Do List, and Rock-Paper-Scissors Site all use JavaScript.

CSS3/Bootstrap 5

Cascading Style Sheets 3 is used to stylize your web pages and ensure that they are appealing to the eye. Bootstrap 5 is a collection of premade CSS3 settings that make it much faster and more efficient to produce a good looking web page. My To-Do List, Paracelsus Computing Site, and Rock-Paper-Scissors Site below use Bootstrap 5, while the Halloween Site Final is entirely CSS3.
